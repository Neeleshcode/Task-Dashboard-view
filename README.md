# 📊 Project Tracker Dashboard

A modern and responsive task management dashboard built using React, TypeScript, and Tailwind CSS. The application enables users to efficiently manage tasks with real-time UI updates, status tracking, and filtering capabilities.

---

## 🚀 Features

* 📌 Create, update, and delete tasks
* 📊 Track task status (To-Do, In Progress, Completed)
* 🔍 Filter and search tasks
* ⚡ Dynamic UI updates for smooth experience
* 🎨 Fully responsive design using Tailwind CSS
* 🧩 Scalable and clean component architecture

---

## 🛠️ Tech Stack

* ⚛️ React
* 🟦 TypeScript
* 🎨 Tailwind CSS
* ⚡ Vite

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Neeleshcode/Task-Dashboard-view.git
```

### 2️⃣ Navigate to project folder

```
cd Task-Dashboard-view
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the project

```
npm run dev
```
